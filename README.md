# 라이프마스터리 어드민

Refine 프레임워크 기반의 관리자 페이지입니다.

## 기술 스택

- **프레임워크**: React 18 + TypeScript
- **빌드 도구**: Vite
- **Admin 프레임워크**: Refine
- **UI 라이브러리**: Ant Design
- **데이터 프로바이더**: REST API
- **라우팅**: React Router v6

## 시작하기

### 개발 서버 실행

```bash
npm run dev
```

개발 서버가 `http://localhost:5174/`에서 실행됩니다.

### 프로덕션 빌드

```bash
npm run build
```

### 빌드 미리보기

```bash
npm run preview
```

## 페이지 네비게이션

### 1. 대시보드 (/)
- 전체 요약 지표: 총 사용자 수, 활성 클래스 수, 진행 중인 챌린지 수, 이번 달 매출
- 최근 활동 피드

### 2. 사용자 관리 (/users)
- **목록**: 검색/필터 (이름, 이메일, 전화번호, 가입일), 테이블
- **상세**: 기본 정보, 구매 이력, 수강 클래스, 참여 챌린지 (탭 구조)

### 3. 클래스 관리 (/classes)
- **목록**: 검색/필터 (클래스명, 기간), 테이블
- **상세**: 클래스 정보, 수강생 관리, 디스코드 채널, 연계 챌린지 (탭 구조)

### 4. 챌린지 관리 (/challenges)
- **목록**: 검색/필터 (챌린지명, 기간, 상태), 테이블, 생성 버튼
- **생성/수정**: 폼 (이름, 기간, 가격, 설명, 요일 설정)
- **상세**: 챌린지 정보, 참여자 관리, 주간 출석 관리 (탭 구조)

### 5. 매출 관리 (/revenue)
- **매출 대시보드**: 기간별 매출 그래프, 상품별 매출 요약, 결제수단별 비율
- **판매 로그**: 검색/필터, 테이블, 엑셀 다운로드
- **수동 매출 추가**: 폼 (날짜, 상품명, 구매자, 금액, 결제수단, 메모)
- **PG 연동 관리**: PG 연동 상태, 데이터 동기화, 동기화 로그
